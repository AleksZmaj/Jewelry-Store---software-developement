# Jewelry-Store---software-developement
This project is a software application developed for managing a jewelry store. The system helps with organizing products, handling sales, tracking customers, and maintaining overall store operations in a user-friendly way.  

Key functionalities include:
- Product management (adding, updating, and removing jewelry items)
- Customer management
- Sales and purchase tracking
- Simple reporting features
- Intuitive interface designed for ease of use

## Technologies
- PHP (Backend programming language)  
- MySQL (Database)  
- HTML, CSS
